# tm1.github.io

Сайт-визитка компании **"ГлобалТаможня"** — профессиональное таможенное оформление в РФ.

## 🚀 Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/tm1/tm1.github.io.git
   cd tm1.github.io
   ```

2. Откройте `index.html` в браузере или разверните на GitHub Pages.

## 🛠 Технологии

- Чистый HTML5 + CSS3
- Адаптивный дизайн (mobile-first)
- SEO-оптимизация (мета-теги, Open Graph, Twitter Cards)
- Доступность (ARIA-атрибуты, семантическая разметка)
- Яндекс.Метрика для аналитики

## 📁 Структура проекта

```
tm1.github.io/
├── index.html       # Главная страница
├── styles.css       # Стили
├── README.md        # Документация
├── favicon.ico      # Иконка сайта (требуется добавить)
└── apple-touch-icon.png  # Иконка для iOS (требуется добавить)
```

## 🔧 Настройка аналитики

Замените идентификатор счётчика Яндекс.Метрики в файле `index.html`:
- Найдите `d5e7f3g2` и замените на ваш ID счётчика
- Замените `00000000` в теге `<noscript>` на тот же ID
